## 1.Exercise: Methods

1. 5.Add and Subtract

You will receive 3 **integers.** Write a method **Sum** to get the sum of the first two integers and **Subtract** method that subtracts the third integer from the result from the Sum method.

### Examples

| **Input** | **Output** |
| --- | --- |
| 23610 | 19 |
| 11730 | -12 |
| 42 58100 | 0 |
