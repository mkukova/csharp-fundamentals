1. 2.Gauss&#39; Trick

Write a program that **sums** all of the **numbers in a list** in the following order:

_first_ **+** _last_, _first + 1_ **+** _last - 1_,_first + 2_ **+** _last_ **-** _2_ **,** … _first + n_, _last - n_.

### Example

| **Input** | **Output** |
| --- | --- |
| 1 2 3 4 5 | 6 6 3 |
| 1 2 3 4 | 5 5 |
