1. 2.English Name of the Last Digit

Write a **method** that returns the **English name** of the last digit of a given number. Write a program that reads an integer and prints the returned value from this method.

### Examples

| **Input** | **Output** |
| --- | --- |
| 512 | two |
| 1 | one |
| 1643 | three |
