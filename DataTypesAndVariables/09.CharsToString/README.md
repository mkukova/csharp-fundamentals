# 1.Lab: Data Types and Variables

1. 9.Chars to String

Write a program that reads 3 lines of input. On each line you get a single character. Combine all the characters into one string and print it on the console.

### Examples

| **Input** | **Output** |
| --- | --- |
| abc | abc |
| %2o | %2o |
| 15p | 15p |
