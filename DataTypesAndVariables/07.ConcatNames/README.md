# 1.Lab: Data Types and Variables

1. 7.Concat Names

Read two names and a delimiter. Print the names joined by the delimiter.

### Examples

| **Input** | **Output** |
| --- | --- |
| JohnSmith-\&gt; | John-\&gt;Smith |
| JanWhite\&lt;-\&gt; | Jan\&lt;-\&gt;White |
| LindaTerry=\&gt; | Linda=\&gt;Terry |
